Session state
Disables Run, Buy, and Sell for every student until resumed. Unlocking reactions, gifting, and End Session still work while paused.
Yield update
—
Immediately ends the current interval (applying normal usage-based updates) and randomly jolts 1-3 reactions on top, so repeat play doesn't become a fully solved, memorize-once pattern. Only has an effect while fluctuating yield is on.
Bonus question
Shown on the confirm screen when a student runs a reaction.
Fluctuating yield
Off: every reaction always yields 100%. On: each reaction starts at 90%, drops the more it's run recently, and recovers (up to 100%) the longer it's left alone.
Gift to all students
Adds to every joined student's heat or inventory -- doesn't touch anything else.
End session
Reads current class records: pays 1st/2nd/3rd for each common goal, and the record holder(s) for each hidden quest. Doesn't reset anything -- safe to run mid-session to check standings, though it will pay out again for whoever's still leading.
Start a new class
Clears every student, the reaction library, and all class records, and resets the market back to a fresh interval. Use this before a new class period, not mid-session.
